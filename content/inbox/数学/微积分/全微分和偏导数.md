# 结论

1. 不妨假设: 
$$
\mathbf{u} = f(x_{1},\dots x_{n})
$$
	则: 
$$
\mathrm{d}u = \frac{ \partial u }{ \partial x_{1} } \mathrm{d}x_{1} + \dots + \frac{ \partial u }{ \partial x_{n} } \mathrm{d} x_{n} = \sum_{i=1}^{n} \frac{ \partial f }{ \partial x_{i} } \mathrm{d} x_{i}  
$$

其中的$\frac{ \partial u }{ \partial x_{n} }$即为偏导数
2. 偏导数的意思就是把其他的变量视为常数, 只对我们所需要导的变量求导即可
3. 存在定理: 若对于u的每一个偏导数在$x_{0}$处都连续, 那么u在$x_{0}$处必定可微! 

# 高阶偏导数

1. 我们记二阶偏导($u$对$x_{i}$, $x_{j}$)的偏导数为: 
$$
\frac{ \partial}{ \partial x_{j} }\left( \frac{ \partial u }{ \partial x_{i} }  \right) = \frac{ \partial^{2} u }{ \partial x_{j} \partial x_{i}}  = f''_{x_{i}x_{j}}
$$
	若$x_{j} = x_{i}$时: 记作$\frac{ \partial^{2} u }{ \partial x_{i}^{2} }$或者$f''_{x_{i}^{2}}$
2. 根据Schwarz定理: 
$$
f''_{xy}(x_{0}, y_{0}) = f''_{yx}(x_{0}, y_{0})
$$
	即交换求偏导的次序不改变最后所求的偏导数值. 该结论具有普适性 ^899ba9
3. 可以进阶为n阶偏导数. 
	若是先对x做k阶偏导数 再对y做n-k阶偏导数, 则记号为:$\frac{ \partial^{n} u }{ \partial y^{n-k} \partial x^{k} }$. 

# 可微映射

对于自变量$\mathbf{x} = (x_{1}, x_{2}, \dots x_{n})$的微元$\mathrm{d}\mathbf{x}$和因变量$\mathbf{u} = (u_{1}, u_{2}, \dots u_{m})$的微元$\mathrm{d} \mathbf{u}$, 其存在一个从$\mathbf{R}^{n}$到$\mathbf{R}^{m}$的映射$f$. 则在满足某一些特殊条件的情况下, $f$是可微的, 且微分为: 
$$
\mathrm{d}\mathbf{u} = \mathbf{J} \mathrm{d}\mathbf{x}
$$
则此处的$J$称为$Jacobi$矩阵, 即该映射在此处的导数. 记作$f'(\mathbf{x_{0}})$. 

若当$\mathbf{x} = (x_{1}, x_{2}, \dots x_{n})^{T}$, $\mathbf{u} = (u_{1}, u_{2}, \dots u_{m})^{T}$时, 映射$f$在$\mathbf{x_{0}}$处可微的充要条件是任何$\mathbf{x_{0}}$均可微. 在对应$Jacobi$矩阵为: 
$$
\begin{pmatrix}
\frac{ \partial f_{1} }{ \partial x_{1} } \frac{ \partial f_{1} }{ \partial x_{2} } \dots \frac{ \partial f_{1} }{ \partial x_{n} }  \\
\frac{ \partial f_{2} }{ \partial x_{1} } \frac{ \partial f_{2} }{ \partial x_{2} } \dots \frac{ \partial f_{2} }{ \partial x_{n} }  \\
\dots \\
\frac{ \partial f_{m} }{ \partial x_{1} } \frac{ \partial f_{m} }{ \partial x_{2} } \dots \frac{ \partial f_{m} }{ \partial x_{n} } 
\end{pmatrix}
$$
规律: 横行f不变 竖行x不变(这样子点乘之后得到的可以保证是正确的, 不会出现无法匹配的情况).
