# 级数性质

2. [[级数乘法]]
3. 级数的收敛点和收敛域
	1. 在取某个特定函数值时该级数收敛, 称该点为函数收敛点
	2. 收敛点的集合称为收敛域
4. 极限函数: 当$n\to \infty$时, 若$S(x) = \lim_{ n \to \infty }S_{n}(x)$那么称$S(x)$为极限函数
5. 余项: $r_{n}(x)=S(x)-S_{n}(x)$
